# FitnessTrackerApp

Android приложение для занятий спортом в домашних условиях.

## Функционал

Приложение содержит 2  основных экрана

1. Список тренировок.
2. Список дней.
3. Список упражнений.
4. Загрузка сета упражнений.
5. Упражнение.
6. Завершение тренировки.

## Инструменты реализации

- Android Studio
- Kotlin
- LiveData
- XML
- CountDownTimer
